# Claude Code Project Rules

## Project Overview
This is a POC environment for rapid development using Claude Code, optimized for Upwork-style freelance projects.

## Tech Stack
- Frontend: Next.js (App Router)
- Language: TypeScript
- Styling: Tailwind CSS
- Tooling: ESLint, Prettier

## Coding Standards
- Follow clean architecture principles.
- Prioritize modular and reusable components.
- Always include a README.md for sub-projects explaining how to run them.

## Git Workflow
- Create descriptive commit messages in English.
- Work within the `basic-agent/` directory unless specified otherwise.
