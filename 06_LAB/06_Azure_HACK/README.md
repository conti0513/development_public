---

## 問題の解き方テンプレ

### Step1：要件を把握

例：

* 外部に固定IPで通信したい
* 内部だけで通信したい

---

### Step2：Azureサービスに変換

例：

* 固定IPで外向き → NAT Gateway
* 内部通信 → Private Endpoint
* L7制御 → Application Gateway

---

### Step3：消去法

* 不適切な選択肢を削除
* 2択まで絞る

---

## 頻出パターン

* 外向き固定IP → NAT Gateway
* 内部限定通信 → Private Endpoint
* L7ロードバランス → Application Gateway
* 複数VNet構成 → Hub-Spoke
* オンプレ接続 → VPN Gateway
