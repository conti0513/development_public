# 🏆 Duolingo English Test (DET) Strategy Guide

### 📊 Test Sections & Evaluation Categories
| Category | Description | Task Examples |
| :--- | :--- | :--- |
| **Literacy** | Reading and Writing | C-Test (fill-in-the-blanks), Real word selection |
| **Comprehension** | Reading and Listening | Read aloud, Dictation |
| **Production** | Writing and Speaking | **Picture Description (1 min)**, Writing/Speaking samples |
| **Conversation** | Listening and Speaking | Interactive listening, Reading responses |

---

### 🛠️ Learning Workflow (Continuous Improvement)
Improve proficiency by treating language acquisition as a software development lifecycle.

1. **Implement (Draft):** Complete tasks within the time limit using "raw" English.
2. **Refactor (AI Review):** Utilize Gemini to elevate the response to **C1/120+ level** (advanced vocabulary & syntax).
3. **Analyze (Diff):** Identify gaps in transitional phrases, tense accuracy, and spelling errors.
4. **Deploy (Repeat):** Reuse and integrate the refined expressions into subsequent practice sessions.

---

### 🚀 Roadmap to 120 (C1 Level)

* **Precision (Accuracy):** Minimize errors in the **Literacy** section to ensure the adaptive algorithm triggers higher-difficulty questions.
* **Sophistication (Production):** * Replace basic verbs: `I think...` → `I firmly believe...` | `good` → `exceptional`
    * Leverage complex structures: Use relative clauses and participial phrases to demonstrate grammatical range to the AI.
* **Mental Adaptivity:** Embrace the "Adaptive Test" nature—higher difficulty is a sign of success. Maintain focus as the questions become more challenging.
